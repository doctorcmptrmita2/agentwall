## Health Check

```
GET https://api.agentwall.io/health
```

Returns basic health status:

```json
{
  "status": "healthy",
  "service": "agentwall-proxy",
  "version": "0.1.0",
  "timestamp": "2026-01-06T12:00:00Z"
}
```

## Detailed Health

```
GET https://api.agentwall.io/health/detailed
```

Returns full system status:

```json
{
  "status": "healthy",
  "service": "agentwall-proxy",
  "version": "0.1.0",
  "environment": "production",
  "checks": {
    "redis": {"status": "healthy"},
    "clickhouse": {"status": "healthy"},
    "openai": {"status": "configured"}
  },
  "config": {
    "max_steps": 30,
    "dlp_mode": "mask",
    "dlp_enabled": true
  }
}
```
