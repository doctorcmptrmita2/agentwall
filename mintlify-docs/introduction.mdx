## Why AgentWall?

Traditional LLM gateways focus on **single requests**. AgentWall focuses on **entire agent runs**.

| Feature | LLM Gateways | AgentWall |
|---------|--------------|-----------|
| Request-level budget | ✅ | ✅ |
| **Run-level budget** | ❌ | ✅ |
| **Loop detection** | ❌ | ✅ |
| **Step counting** | ❌ | ✅ |
| DLP/PII protection | Some | ✅ |
| Multi-provider | ✅ | ✅ |

## Quick Example

Replace your OpenAI base URL and you're protected:

```python
from openai import OpenAI

# Before: Direct to OpenAI
client = OpenAI(api_key="sk-...")

# After: Through AgentWall
client = OpenAI(
    api_key="sk-...",
    base_url="https://api.agentwall.io/v1"
)

# That's it! Your agent is now protected.
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
```
